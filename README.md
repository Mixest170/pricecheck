# ตัวไหนคุ้มกว่า? · เครื่องเทียบราคาสินค้า

เทียบราคาต่อหน่วยของสินค้า 2 ตัว แล้วบอกว่าตัวไหนถูกกว่า ถูกกว่ากี่ % และต่างกันมากแค่ไหน
มี 2 สูตร:

1. **ราคา + ปริมาณ** — รู้ราคาและปริมาณรวม (เช่น 135 บาท / 2000 มล.)
2. **ราคา + จำนวน + ขนาด** — ของขายเป็นแพ็ก (เช่น 6 ขวด ขวดละ 500 มล.) ระบบคูณ จำนวน × ขนาด ให้เอง

Stack: Vite + React + Tailwind CSS v4

## เปลี่ยนรูปโลโก้ฟุตเตอร์
รูปฝังอยู่ในไฟล์ `src/App.jsx` แล้ว เป็น data URI ในตัวแปร `LOGO` ด้านบนสุดของไฟล์
ถ้าอยากเปลี่ยนรูป แปลงรูปใหม่เป็น base64 data URI แล้ววางแทนค่า `LOGO` ได้เลย (ไม่ต้องแยกไฟล์รูป)

## รันในเครื่อง (local)
ต้องมี Node.js v18+ (เช็คด้วย `node -v`)

```bash
npm install      # ติดตั้งครั้งแรกครั้งเดียว
npm run dev      # เปิด dev server -> http://localhost:5173
```

## ขึ้น Vercel

### ทาง A — ผ่าน GitHub (แนะนำ ง่ายสุด ไม่ต้องใช้ command line)
1. สร้าง repo ใหม่บน GitHub แล้วอัปโหลด/ push ไฟล์ "ข้างใน" โฟลเดอร์นี้ทั้งหมด
   (ให้ `package.json` อยู่ที่ระดับบนสุดของ repo — อย่าอัปโหลดทั้งโฟลเดอร์ครอบ)
2. ไป vercel.com -> Sign up / Login ด้วย GitHub
3. Add New… -> Project -> เลือก repo นี้ -> Import
4. Vercel จะตรวจเจอว่าเป็น Vite เอง ไม่ต้องตั้งค่าอะไร
5. กด Deploy -> ได้ลิงก์เว็บ
6. หลังจากนี้ push ทีไร Vercel deploy ใหม่ให้อัตโนมัติ

### ทาง B — ผ่าน Vercel CLI
```bash
npm i -g vercel
vercel
vercel --prod
```

## แก้สูตร / ปรับข้อความ
ทุกอย่างอยู่ในไฟล์เดียว: `src/App.jsx`
- ตรรกะการเทียบราคาอยู่ในฟังก์ชัน `compare()` และ `verdictBand()` (เกณฑ์ % ที่บอกว่า "ถูกกว่านิดเดียว / ชัดเจน / มาก")
- ลิงก์ Facebook อยู่ในตัวแปร `FB_URL` บนสุดของไฟล์
